# Minimal Paint

Minimal Paint is a wonderful drawing program based on [AcrylicPaint](https://github.com/anupam1525/AcrylicPaint).
Simple, neat while full of possibility.
Get pleasure from drawing on the screen of your phone! Draw everything that you want and then share your masterpieces with friends.
